# MyFirstCIProject

## Description
This is my first project to learn about Continuous Integration and Continuous Deployment (CI/CD). The project aims to familiarize myself with the basics of setting up CI/CD pipelines using GitHub.

## Goals
- Understand the fundamentals of CI/CD.
- Learn how to create and manage CI/CD pipelines.
- Automate testing and deployment processes.
- Gain hands-on experience with YAML for configuration.
